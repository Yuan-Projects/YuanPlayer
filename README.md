YuanPlayer
==========

Features:
 * Play
 * Pause
 * Toggle Play
 * Stop
 * Mute
 * Unmute
 * Toggle Mute
 * Volume++
 * Volume--
 * Duration
 * Current Time
 * Live stream autio support
 * Toggle Loop

YuanPlayer Compatibility

 * Firefox 27 (Windows, Mac OS X)
 * Internet Explorer 11 (Windows)
 * Internet Explorer 10 (Windows)
 * Internet Explorer 9 (Windows)
 * Safari 7.0.2(Mac OS X)
